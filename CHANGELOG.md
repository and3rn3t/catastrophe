  - AHumanNPC - AI human NPCs
  - ACatastropheGameMode - Game mode management
- Unreal Engine 5.3 project setup
- Basic project structure
- Documentation
  - README.md with project overview
  - SETUP.md for quick setup
  - CONTRIBUTING.md for contribution guidelines
  - ROADMAP.md for development planning
  - DESIGN.md for visual design guide
  - QUICKSTART.md for quick reference
- Config files for Unreal Engine
  - DefaultEngine.ini
  - DefaultGame.ini
  - DefaultInput.ini
  - DefaultEditor.ini
