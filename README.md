# CATastrophe: A Mischief Simulator

A playful sandbox game where you play as a mischievous cat causing mayhem around the house! Inspired by Untitled Goose Game, this game lets you knock over vases, destroy furniture and curtains, steal food, all while your humans might be watching.

## Game Features

- **Play as a Cat**: Control a cute but destructive feline
- **Realistic Physics**: Objects react naturally to your mischief
- **Interactive Environment**: Knock over vases, scratch furniture, push items off tables
- **Stealth Mechanics**: Avoid getting caught by watchful humans
- **Dynamic Interactions**: Multiple ways to interact with objects
- **Food Stealing**: Sneak food when humans aren't looking

## How to Play

**Controls:**
- **Arrow Keys** or **WASD**: Move your cat around
- **Spacebar**: Interact with nearby objects (knock over, scratch, steal)
- **Shift**: Sprint (but makes more noise!)

**Objective:**
Cause as much mischief as possible while avoiding being caught by humans. Different objects give different points when destroyed or knocked over!

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/and3rn3t/catastrophe.git
cd catastrophe
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The game will open in your browser at [http://localhost:3000](http://localhost:3000).

## Available Scripts

### `npm start`
Runs the app in development mode with hot reloading.

### `npm test`
Launches the test runner in interactive watch mode.

### `npm run build`
Builds the app for production to the `build` folder.
